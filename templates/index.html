🧠 EEG Emotion Classifier

Upload an EEG recording to detect emotional state with 89% accuracy

🎯 Universal EEG Adapter

✓ Works with any EEG device (Muse, Natus, EGI, BioSemi, g.tec, Neuroscan, Nihon Kohden, Brain Products)

✓ Supports raw .edf files directly from clinical machines

✓ 21 wave features across Delta, Theta, Alpha, Beta, Gamma bands

📊 89.7% Accuracy

Optimized XGBoost ensemble with [1,2,5] weights

💖 Open for Improvement & Doctor Validation

This model is trained on limited data (2 subjects). To make it truly generalizable across different people, we need more EEG data with known emotional states.

✅ Train the model on more subjects
✅ Improve cross-subject validation
✅ Make emotion detection more accurate for everyone
